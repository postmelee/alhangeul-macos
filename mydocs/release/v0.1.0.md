`v0.1.0`은 알한글의 첫 공식 public release다. Finder Quick Look preview, Finder thumbnail, WKWebView 기반 HWP/HWPX viewer/editor, HWP 저장/다른 이름 저장, PDF 내보내기, macOS 공유, 인쇄, Sparkle 업데이트 확인 경로를 제공한다.

## 배포 결정

- public 사용자 배포 기준은 Developer ID signed + notarized DMG로 정했다.
- unsigned app/DMG, ad-hoc signed app/DMG, rehearsal DMG, 개발용 zip은 public release asset 또는 Homebrew Cask URL로 사용하지 않는다.
- 앱 filesystem bundle name은 `Alhangeul.app`으로 유지한다.
- public DMG 산출물명은 `alhangeul-macos-<version>.dmg` 형식을 사용한다.
- 사용자 표시명은 localization으로 한국어 `알한글`, 영어 `Alhangeul`을 사용한다.

비밀이 아닌 운영 환경 식별자는 [`../tech/release_environment.md`](../tech/release_environment.md)에 분리한다.

- Quick Look/Thumbnail extension 상태 진단
- Sparkle 업데이트 확인 경로

## 알려진 제한 사항

- 앱 화면의 viewer/editor와 Finder Quick Look/thumbnail, PDF 내보내기, 인쇄는 서로 다른 렌더링 경로를 사용할 수 있다.
- Quick Look/Thumbnail smoke 통과는 extension 등록과 기본 렌더 성공을 뜻하며, 모든 문서가 앱 화면과 같은 시각 결과로 보인다는 보장은 아니다.
- HWPX 문서는 직접 저장이 제한되어 HWP export 경로를 사용한다.
- 손상, 대용량, 미지원 문서 fallback은 앱과 extension이 멈추지 않도록 하는 안전장치이며, 파일 복구나 부분 렌더링을 보장하지 않는다.
- native renderer의 style, image effect/fill, text layout, RawSvg/OLE 등 parity 개선은 Swift native viewer 방향에서 계속 다룬다.
