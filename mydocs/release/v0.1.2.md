## 알려진 제한 사항과 후속 이슈

- Quick Look preview와 Finder thumbnail은 native renderer 경로를 사용하고, 앱 viewer/editor는 WKWebView 기반 `rhwp-studio` 경로를 사용한다.
- build별 launch maintenance는 `NSWorkspace.noteFileSystemChanged`로 Finder/Quick Look 재평가를 유도하지만, 전역 Quick Look cache 삭제를 보장하지 않는다.
- 최근 문서 후보 밖 파일의 오래된 thumbnail cache는 그대로 남을 수 있다.
- HWPX 문서는 현재 직접 저장이 제한되어 HWP export 경로를 사용한다.
- public DMG SHA256, Sparkle appcast signature, GitHub Release latest 상태는 Stage 5/public 배포 후 확정한다.
