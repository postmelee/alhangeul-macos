# Issue #11 단계 3 완료 보고서

## 수행 내용

- `AGENTS.md`의 `PR 처리 규칙 (pr/)` 목차를 `외부 기여자 PR 처리 (mydocs/pr/)`로 변경했다.
- 해당 목차에는 외부 기여 PR 검토에 필요한 최소 규칙만 남겼다.
- `mydocs/manual/pr_process_guide.md`에서 로컬 작업자 PR 생성/본문 작성 절차를 제거하고 외부 기여 PR 검토 절차 중심으로 정리했다.

## 판단

로컬 작업자 PR 생성 규칙은 `AGENTS.md`의 `Git 워크플로우 > PR 생성`에 이미 정의돼 있다. 외부 기여 PR 처리와 섞으면 적용 대상이 모호해지므로 목차 목적을 분리하는 편이 운영상 안전하다.

## 결과

`AGENTS.md`는 빠른 가이드 역할을 유지하면서도 적용 대상을 명확히 했고, `pr_process_guide.md`는 외부 기여 PR 검토 문서로 역할을 고정했다.
