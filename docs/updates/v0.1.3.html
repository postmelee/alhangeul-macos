알한글 v0.1.3

업스트림 rhwp v0.7.12를 반영해 HWP/HWPX 문서 열기와 viewer asset 기준을 갱신한 패치 릴리즈입니다.

DMG 다운로드 GitHub Release 업데이트 목록

전체 요약

  • rhwp core와 bundled rhwp-studio를 v0.7.12 기준으로 갱신했습니다.
  • 업스트림의 HWP/HWPX parser, viewer, WMF, HWP3, 렌더링 관련 수정이 알한글 앱과 Finder 통합 경로에 포함됩니다.
  • 앱 본체, Quick Look preview, Finder thumbnail extension의 release version을 0.1.3 (9)로 맞췄습니다.
  • 공식 DMG는 Intel Mac과 Apple Silicon Mac에서 같은 파일을 사용하는 signed/notarized universal DMG 기준을 유지합니다.

포함된 rhwp 변화

v0.1.3은 rhwp v0.7.12 기준의 core와 bundled rhwp-studio를 사용합니다. 앱 내부 provenance 기준 commit은 1899ef9bc2dfd1c6c0c4d18b192d253a2d0a1fb5입니다.

  • 업스트림 tag 메시지 기준으로 여러 PR의 parser/viewer 결함 수정과 WMF, HWP3, LTO 관련 개선이 포함됩니다.
  • bundled rhwp-studio Web/WASM asset도 같은 v0.7.12 commit에서 다시 빌드했습니다.
  • Rust FFI에는 native PNG 렌더링용 rhwp_render_page_png symbol이 추가됐고, 앱 viewer와 Finder extension 경로는 새 bridge artifact 기준으로 검증했습니다.

알한글 앱 변화

  • About 창과 내부 provenance 기준이 rhwp v0.7.12를 표시할 수 있도록 bundled manifest를 갱신했습니다.
  • release rehearsal/publish workflow의 기본 입력을 v0.1.3, 직전 공개 릴리즈 v0.1.2, expected rhwp v0.7.12 기준으로 정리했습니다.
  • third-party notices와 release record가 앱 bundle에 포함된 core/studio 기준을 같은 commit으로 가리키도록 맞췄습니다.

알려진 한계

  • Quick Look과 Finder 썸네일은 native 렌더 경로를 사용하고, 앱 화면은 WKWebView 기반 rhwp-studio 경로를 사용합니다.
  • 일부 문서에서는 글꼴, 배치, 페이지 표현이 원본 한글 프로그램과 다르게 보일 수 있습니다.
  • HWPX 문서는 직접 저장이 제한되어 HWP 내보내기 경로를 사용합니다.

설치와 업데이트

DMG 파일을 내려받아 앱을 Applications 폴더로 옮긴 뒤, 앱을 한 번 실행합니다. v0.1.3 공식 DMG는 Intel Mac과 Apple Silicon Mac에서 같은 파일을 사용합니다.

Homebrew Cask를 사용하는 경우 아래 명령으로 같은 signed/notarized universal DMG를 설치할 수 있습니다.

brew install --cask postmelee/tap/alhangeul

설치한 앱에서는 메뉴 막대의 알한글 > 업데이트 확인...을 선택해 새 버전을 확인할 수 있습니다.
