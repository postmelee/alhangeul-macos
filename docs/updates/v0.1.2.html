알한글 v0.1.2

업스트림 rhwp v0.7.11을 반영하고, Sparkle 업데이트 후 기존 Finder 썸네일이 오래 남는 상황을 줄이기 위한 첫 실행 maintenance와 About 창의 bundled rhwp provenance 표시를 추가한 패치 릴리즈입니다.

DMG 다운로드 GitHub Release 업데이트 목록

주요 변경

  • rhwp core와 bundled rhwp-studio를 v0.7.11 기준으로 갱신했습니다.
  • About 창에서 앱에 포함된 rhwp release tag와 commit provenance를 확인할 수 있습니다.
  • 새 build를 처음 실행할 때 Quick Look/Thumbnail extension 등록을 갱신하고, 최근에 열었던 HWP/HWPX 문서의 Finder thumbnail refresh를 요청합니다.
  • 업데이트 maintenance는 build별 marker를 사용하므로 같은 build를 반복 실행할 때 매번 registration refresh를 반복하지 않습니다.
  • 앱 소유 HWP/HWPX UTI를 com.postmelee.alhangeul.* 기준으로 고정하고 Hancom 계열 UTI도 함께 처리합니다.
  • 문서가 열린 뒤 발생하는 Web viewer runtime 오류는 문서를 가리는 fatal 화면 대신 상단 banner로 안내합니다.
  • 공식 DMG는 Intel Mac과 Apple Silicon Mac에서 같은 파일을 사용하는 signed/notarized universal DMG 기준을 유지합니다.

포함된 rhwp

v0.1.2는 rhwp v0.7.11 기준의 core와 bundled rhwp-studio를 사용합니다. 앱 내부 provenance 기준 commit은 a9dcdee32b17a7f9a20c609a5ed547e62fb8ebae입니다.

알려진 한계

  • Quick Look과 Finder 썸네일은 native 렌더 경로를 사용하고, 앱 화면은 WKWebView 기반 rhwp-studio 경로를 사용합니다.
  • 기존 Finder cache가 남아 있는 일부 파일은 최근 문서 후보 밖에 있거나 Finder 정책상 즉시 갱신되지 않을 수 있습니다.
  • 일부 문서에서는 글꼴, 배치, 페이지 표현이 원본 한글 프로그램과 다르게 보일 수 있습니다.
  • HWPX 문서는 직접 저장이 제한되어 HWP 내보내기 경로를 사용합니다.

설치와 업데이트

DMG 파일을 내려받아 앱을 Applications 폴더로 옮긴 뒤, 앱을 한 번 실행합니다. v0.1.2 공식 DMG는 Intel Mac과 Apple Silicon Mac에서 같은 파일을 사용합니다. 첫 실행 후 Quick Look preview와 Finder thumbnail extension이 등록되고 최근 문서 thumbnail refresh가 요청될 수 있습니다.

Homebrew Cask를 사용하는 경우 아래 명령으로 같은 signed/notarized universal DMG를 설치할 수 있습니다.

brew install --cask postmelee/tap/alhangeul

설치한 앱에서는 메뉴 막대의 알한글 > 업데이트 확인...을 선택해 새 버전을 확인할 수 있습니다.
