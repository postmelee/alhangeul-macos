알한글 v0.1.1

설치본에서 확인된 창 확대, Finder 썸네일, Quick Look/Thumbnail 크래시 문제를 수정하고, Intel Mac과 Apple Silicon Mac을 모두 지원하는 단일 universal DMG 기준을 더 분명하게 다듬은 패치 릴리즈입니다.

DMG 다운로드 GitHub Release 업데이트 목록

주요 변경

  • 문서를 연 상태에서 창을 확대하거나 크기를 바꿀 때 WebView 오류 화면으로 전환되던 문제를 수정했습니다.
  • 공식 설치본에서 Finder 썸네일 생성이 멈출 수 있던 문제를 줄였습니다.
  • 일부 설치본에서 Quick Look preview와 Finder thumbnail extension이 렌더링 중 종료되어 fallback 카드나 깨진 썸네일이 보이던 문제를 수정했습니다.
  • 앱 본체와 Quick Look/Thumbnail extension 실행 파일을 arm64 + x86_64 universal 기준으로 검증합니다.
  • DMG 설치 창에 Applications로 드래그하는 흐름과 첫 실행 안내가 보이도록 개선했습니다.
  • 릴리즈 노트와 업데이트 페이지가 설치, 첫 실행, Sparkle 업데이트 확인, 검증 정보를 같은 기준으로 안내하도록 정리했습니다.

포함된 rhwp

v0.1.1은 rhwp v0.7.10 기준의 core와 bundled rhwp-studio를 사용합니다. 세부 provenance는 알한글 GitHub Release와 릴리즈 기록에서 확인할 수 있습니다.

알려진 한계

  • Quick Look과 Finder 썸네일은 native 렌더 경로를 사용하고, 앱 화면은 WKWebView 기반 rhwp-studio 경로를 사용합니다.
  • 일부 문서에서는 글꼴, 배치, 페이지 표현이 원본 한글 프로그램과 다르게 보일 수 있습니다.
  • HWPX 문서는 직접 저장이 제한되어 HWP 내보내기 경로를 사용합니다.

설치와 업데이트

DMG 파일을 내려받아 앱을 Applications 폴더로 옮긴 뒤, 앱을 한 번 실행합니다. v0.1.1 공식 DMG는 Intel Mac과 Apple Silicon Mac에서 같은 파일을 사용합니다. 첫 실행 후 Quick Look preview와 Finder thumbnail extension이 등록될 수 있습니다.

설치한 앱에서는 메뉴 막대의 알한글 > 업데이트 확인...을 선택해 새 버전을 확인할 수 있습니다.

알한글

Mac을 위한 HWP/HWPX 문서 미리보기 및 편집 앱입니다. 한글 파일이 더 이상 낯선 파일로 남지 않도록 만듭니다.
