알한글 v0.1.0

Mac에서 HWP/HWPX 문서를 미리보고, 찾고, 열어보는 첫 공식 릴리즈입니다.

DMG 다운로드 GitHub Release 업데이트 목록

주요 기능

  • Finder Quick Look에서 HWP/HWPX 문서 첫 페이지를 빠르게 확인합니다.
  • Finder 썸네일로 문서를 열기 전에 첫 페이지를 구분합니다.
  • 앱에서 HWP/HWPX 문서를 열고 텍스트 중심의 기본 편집 흐름을 사용합니다.
  • PDF 내보내기, macOS 공유, 인쇄 흐름을 제공합니다.
  • 앱 메뉴에서 업데이트 확인...을 실행해 새 버전 안내를 받을 수 있습니다.

포함된 rhwp

v0.1.0은 rhwp v0.7.10 기준의 core와 bundled rhwp-studio를 사용합니다. 세부 provenance는 알한글 GitHub Release와 릴리즈 기록에서 확인할 수 있습니다.

알려진 한계

  • Quick Look과 Finder 썸네일은 native 렌더 경로를 사용하고, 앱 화면은 WKWebView 기반 rhwp-studio 경로를 사용합니다.
  • 일부 문서에서는 글꼴, 배치, 페이지 표현이 원본 한글 프로그램과 다르게 보일 수 있습니다.
  • HWPX 문서는 직접 저장이 제한되어 HWP 내보내기 경로를 사용합니다.

설치와 업데이트

DMG 파일을 내려받아 앱을 Applications 폴더로 옮겨 설치합니다. 이후 버전은 앱 메뉴의 알한글 > 업데이트 확인...에서 확인할 수 있습니다.

릴리즈 DMG와 checksum은 GitHub Release v0.1.0에서 함께 확인할 수 있습니다.

알한글

Mac을 위한 HWP/HWPX 문서 미리보기 및 편집 앱입니다. 한글 파일이 더 이상 낯선 파일로 남지 않도록 만듭니다.
